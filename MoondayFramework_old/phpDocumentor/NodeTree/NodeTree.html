Class NodeTree

Requires PhpDtObject class Node Tree class creates multidimensional associative array from database table that is defined as explained in Node class. Then class is used to create standard hierachically structured html list. With additional javascript class finally creates all necessary codes for manipulating tree visually.

Usage example: $oNT = new NodeTree(...); $oNT->build_tree($aParam); echo $oNT->get_js_contextual_menu(...); echo $oNT->geT_ul_tree(...);

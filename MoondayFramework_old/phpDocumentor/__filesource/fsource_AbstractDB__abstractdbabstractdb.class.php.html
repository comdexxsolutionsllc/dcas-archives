    66. 

  66.     * 
    67.

    68. 

  68.     * 
    69. 

  69.     *  A string composed of the connection arguments and optional or driver specific options in the form
    70. 

  70.     *  of <b>Type</b>://<b>Username</b>:<b>Password</b>@<b>Hostname</b>:<b>Port</b>/<b>Database</b>?<b>Options</b>/option1=value1&<b>Options</b>/option2=value2
    71. 

  71.     * 
    72. 

  72.     *  e.g. $args = array("ConnectionString" => "mysql://root:pass@localhost/MyDatabase?Options/Persistent=1");
    73. 

  73.     * 
    74. 

  74.     *  or
    75. 

  75.     * 
    76. 

  76.     *  Type: The type of database to connect to. AbstractDB is currently only distributed with a MySQL driver.
    77. 

  77.     * 
    78. 

  78.     *  Username: The database username.
    79. 

  79.     * 
    80. 

  80.     *  Password: The database password.
    81. 

  81.     * 
    82. 

  82.     *  Hostname: The database hostname or IP Address.
    83. 

  83.     * 
    84. 

  84.     *  Database: The name of the database.
    85. 

  85.     * 
    86. 

  86.     *  Options: An associative array of optional or driver specific options. See individual driver
    87. 

  87.     *  documentation for a potential list of driver specific options. The value for Port should be
    88. 

  88.     *  specified in this list, otherwise the default port for the database type will be used.
    89. 

  89.     * 
    90. 

  90.     *  e.g. $args = array("Type" => "mysql", "Username" => "root", "Password" => "pass", "Hostname" => "localhost", "Database" => "MyDatabase", "Options" => array("Persistent" => 1));
    91. 

  91.     * 
    92. 

  92.     *  Type and Database are required parameters and must be specified.
    93.
